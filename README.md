# SQLcraft — Learn SQL Interactively

A free, open-source SQL learning app that runs entirely in your browser.
No install, no account, no server needed.

🌐 **Live app:** https://yourusername.github.io/sqlcraft

## Features
- 21 lessons across Beginner, Intermediate and Advanced levels
- 103 hands-on exercises with live SQL execution
- 4 SQL dialects: SQLite, MySQL, PostgreSQL, T-SQL
- Progressive hint system — solution unlocks after 4 wrong attempts
- SQL cheat sheet, progress tracking, dialect syntax comparisons
- 100% offline — SQLite runs in the browser via sql.js

## How to use
Just open the link above. Everything runs in your browser using SQLite.

## Connecting MySQL / PostgreSQL / T-SQL
The app supports live connections to real databases via a lightweight proxy.
See the [How to Connect](#) section inside the app for setup instructions.

## Tech stack
- Vanilla HTML, CSS, JavaScript — zero frameworks
- [sql.js](https://github.com/sql-js/sql.js) — SQLite compiled to WebAssembly
- Google Fonts (Syne, JetBrains Mono, DM Sans)
