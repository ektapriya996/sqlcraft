Connect to MySQL
Connection String Format:
MySQL: mysql://user:pass@host:3306/dbname
PostgreSQL: postgresql://user:pass@host:5432/dbname
T-SQL: mssql://user:pass@host:1433/dbname
⚠️ Requires a backend proxy to be running. See the "How to Connect" tab or the project README for setup instructions. All exercises work fully offline on SQLite without any backend.
SQLcraft is fully functional offline using SQLite.
All 21 lessons and 103 exercises work with zero setup.

To connect MySQL / PostgreSQL / T-SQL:
You need a lightweight proxy server that forwards queries from the browser to your database. A reference proxy is included in the GitHub repo:

cd proxy && npm install && node server.js

The proxy listens on localhost:4000/api/query and accepts:
POST { dialect, sql, connection }
Returns: { columns, rows } or { error }

Quick options to get a free database:
• MySQL: PlanetScale (free tier)
• PostgreSQL: Neon.tech (free tier)
• T-SQL: Azure SQL (free tier)

SQL Cheat Sheet
Core Queries
SELECTFetch columns from a table
WHEREFilter rows by condition
ORDER BYSort ASC or DESC
DISTINCTRemove duplicate rows
LIMIT nCap rows returned (not T-SQL)
ASAlias a column or table
Aggregates & Grouping
COUNT(*)Count all rows
SUM / AVGTotal / average of a column
MIN / MAXSmallest / largest value
GROUP BYGroup rows for aggregation
HAVINGFilter groups (not rows)
Joins
INNER JOINRows matching in both tables
LEFT JOINAll left rows + matched right
ONJoin condition: t1.id = t2.fk
Advanced
WITH … ASCommon Table Expression (CTE)
CASE WHENConditional logic in SELECT
OVER()Window function frame
PARTITION BYReset window per group
ROW_NUMBERUnique sequential rank
RANK / DENSE_RANKRank with / without gaps
LAG / LEADPrevious / next row value
DML & Transactions
INSERT INTOAdd new rows
UPDATE … SETModify existing rows
DELETE FROMRemove rows
BEGIN / COMMITWrap in a transaction
ROLLBACKUndo transaction changes
Dialect Quick Reference
MySQLAUTO_INCREMENT, GROUP_CONCAT, START TRANSACTION
PostgreSQLSERIAL, STRING_AGG, COALESCE, ILIKE
T-SQLIDENTITY(1,1), TOP n, ISNULL, BEGIN TRANSACTION
